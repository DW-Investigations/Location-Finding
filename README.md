This repository contains practical investigations demonstrating the use of Open-Source Intelligence (OSINT) techniques for image geolocation and digital analysis.

## Case 1: Image Geolocation – Abuja, Nigeria

### Objective
Determine the location of an accommodation image posted on social media.

### Methodology
- Reverse image search (Google)
- Cross-referencing with independent sources
- Visual analysis of environmental and structural features

### Key Findings
- Location identified: Abuja, Central Business District (Leventis Close)
- Confidence Level: High

### Skills Demonstrated
- OSINT techniques
- Image geolocation
- Visual pattern recognition
- Evidence validation
